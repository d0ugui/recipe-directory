## 📖 Projeto

- O projeto faz parte de um curso de React.js feito na Udemy. Na aplicação utilizamos alguns React Hooks como useState e useEffect, além disso, também foram abordados alguns temas como componentização e imutabilidade. 

## 🧪 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias e bibliotecas:

- [React](https://reactjs.org)

## 🚀 Como executar

Clone o projeto e acesse a pasta do mesmo.

```bash
$ git clone https://github.com/d0ugui/magic-memory.git
$ cd magic-memory
```

Para iniciá-lo, siga os passos abaixo:

```bash
# Instalar as dependências
$ yarn or npm i

# Iniciar o projeto
$ yarn start or npm start
```

O app estará disponível no seu browser pelo endereço http://localhost:3000/
